# AI-Driven Retail Sales & Customer Analytics Dashboard

## Overview
This project is a complete retail analytics solution developed using Python, MySQL, and Power BI. The system analyzes retail sales data, performs customer and sales analysis, predicts future sales using machine learning, and presents insights through interactive Power BI dashboards.

The project follows an end-to-end data analytics workflow where Python is used for data cleaning, preprocessing, exploratory data analysis, and machine learning; MySQL is used for data storage and SQL-based analysis; and Power BI is used for creating professional business intelligence dashboards.

## Key Features
- Data cleaning and preprocessing using Python
- SQL database integration using MySQL
- Sales and profit analysis
- Customer segmentation and behavior analysis
- Top-selling products and region-wise performance analysis
- Sales forecasting using Linear Regression
- Interactive Power BI dashboards
- KPI monitoring and business insights

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- MySQL
- SQL
- Power BI
- Matplotlib / Seaborn

## Project Workflow
Dataset → Python Cleaning & Analysis → MySQL Database → SQL Queries → Power BI Dashboard → Sales Forecasting & Business Insights

## Business Objectives
- Identify high-performing products and regions
- Analyze customer purchasing behavior
- Improve decision-making using data visualization
- Forecast future sales trends using machine learning

## Outcomes
This project demonstrates practical skills in data analytics, business intelligence, machine learning, and database management using real-world retail data.
